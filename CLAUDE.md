# Exp — Lab di esperimenti

Questa è la root del progetto. Contiene solo:
- `index.html` — home page con il menu degli esperimenti
- Una cartella per ogni esperimento (es. `fretboard/`, `altro-esperimento/`)

## Regola fondamentale

**Non modificare file in questa cartella root.** Ogni esperimento vive dentro la propria sottocartella. Quando lavori su un esperimento, apri la sua cartella e lavora esclusivamente lì.

Se hai bisogno di aggiungere un nuovo esperimento:
1. Crea una nuova sottocartella con un nome descrittivo
2. Aggiungi un `CLAUDE.md` dentro quella cartella
3. Aggiorna il menu in `index.html` aggiungendo la voce corrispondente
