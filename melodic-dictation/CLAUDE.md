# Melodic Dictation — Ear Training App

App di ear training: genera sequenze di note casuali e le riproduce, l'utente le canta/suona mentre l'accordatore mostra la vicinanza alla nota target.

## Scope

Lavora **solo dentro questa cartella** (`melodic-dictation/`). Non toccare file fuori da qui, tranne per aggiornare il menu in `../index.html` se necessario.

## File

- `index.html` — applicazione completa (single-file, no build step)
